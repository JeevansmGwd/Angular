Install node js and check node version -->  (node -v)

# Update npm and nodejs
# npm install -g npm stable

To install node Angular CLI globally -->  npm install -global @angular/cli@latest

To check angularCLI is installed --> ng version

Move to the folder where you want to create a project using cd

create a new project --> ng new ProjectName
cd to newly created project workspace


ng serve --> compiles angular project and generates bundle files and generate liveserver.

open the project folder in vscode.
go to ./src/app/app.component.html

Remove the content in the app.component.html and the page will be blank
Add the command in the html file -- > <h2>Welcome to {{title}}</h2>
Refresh the page

go to app.component.ts  we can see Appcomponent function and title
add other parameters and call in html
